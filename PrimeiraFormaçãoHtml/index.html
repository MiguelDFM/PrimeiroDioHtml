HTML, o que é, e porque é tão importante ?

A html é importante por ser uma linguagem de marcação e por definir a estrutura e o conteúdo
das páginas na web, responsável pela apresentação e definição tanto de texto,
como imagens, vídeos, formulários e outros.






Breve história da HTML

A Html surgiu em 1991 por Tim Berners-Lee , um físico inglês,
e é ele considerado o pai da web,
a primeira versão era bem simples,
mas ao longo dos anos foi envoluindo para atender às novas necessidades do mundo da web.




Como funciona o Html e o que são Tags ?

(Voltar ao início)

As tags são elementos que indicam ao navegador o tipo de conteúdo a ser exibido, como por exemplo as tags "h" que são de título sendo a maior a de nível 1,
ou a tag "img" para indicar o seguimento de uma imagem.
O código de Html é lido pelo navegador, e a partir das tags, intepreta o conteúdo das páginas e o exibe de acordo com as instruções.
A html pode ser uma linguagem simples, mas muito importante para qualquer desenvolvedor web.

Principais razões para aprender html:

(Voltar ao início)
  1. É a linguagem de marcação mais popular do mundo
  2. É uma linguagem gratuita e de código aberto para todos
  3. Fácil de aprender, mesmo para iniciantes
  4. Flexibilidade
